## About

This repository contains Netlify-hosted tools for Wiley University's Institutional Research, Institutional Effectiveness, and Institutional Advancement offices.

### Surveys
Each survey folder includes:
- Survey deployment information
- Live survey link
- Response data link (Google Sheets)
- Survey documentation

### Grants Dashboard
The Grants Dashboard folder includes:
- `index.html` - Full interactive dashboard
- `Wiley_Grants_Database.csv` - Source data (84 grants)
- `Strategic_Plan_External_Funding.md` - 5-year strategic plan
- `Database_Utilization_Strategy.md` - Operational procedures

## Technology Stack

- **Hosting:** Netlify
- **Backend:** Google Apps Script (surveys)
- **Data Collection:** Google Sheets (surveys)
- **Dashboard:** Single-file HTML/CSS/JavaScript application
- **Cost:** Free tier
