## About

This repository contains Netlify-hosted surveys for Wiley University's Institutional Research and Institutional Effectiveness office. Each survey folder includes:

- Survey deployment information
- Live survey link
- Response data link (Google Sheets)
- Survey documentation

## Technology Stack

- **Hosting:** Netlify
- **Backend:** Google Apps Script
- **Data Collection:** Google Sheets
- **Cost:** Free tier (unlimited responses)
